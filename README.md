# Li Bai Webpage

GitHub Pages에 바로 올릴 수 있는 정적 웹페이지입니다.

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 폰트, 레이아웃, 전환 효과, 점멸/노이즈 효과
- `script.js`: 화면 전환 로직
- `assets/moon.png`: 달 이미지

## 동작

- 메인 화면에서 `Li Bai` 텍스트를 누르면 시구 반복 화면으로 전환됩니다.
- 시구 반복 화면의 `← Main` 버튼을 누르면 메인 화면으로 돌아갑니다.
- 메인 화면에서 달 이미지를 누르면 인물 선택 화면으로 전환됩니다.
- 인물 선택 화면의 달 이미지는 불투명도 30%로 표시됩니다.
- `미수희`, `연`, `영`, `정서` 텍스트는 각각 클릭 가능한 버튼입니다.
- 각 인물 페이지는 현재 빈 페이지로 마련되어 있습니다.
- `Esc` 키를 누르면 이전 단계로 돌아갑니다.
